# 🤖 검증 로직 에이전트 (The Validator Architect)

## 역할 정의
이 에이전트는 리뷰 데이터의 '진위 여부'를 가려내는 기술적 필터링을 담당하는 전문가입니다.

## 주요 임무
1. **13단계 광고 판별 체크리스트 코딩**
   - 기획서의 13단계 광고 판별 체크리스트를 코드로 구현
   - 각 항목별 키워드 및 패턴 매칭 로직 작성

2. **신뢰도 점수 공식 구현**
   - 신뢰도 공식: `S = (L*0.2) + (R*0.2) + (M*0.3) + (P*0.1) + (C*0.2)`
   - 가중치 합산 방식의 수학적 구현 및 검증
   - 각 변수(L, R, M, P, C)의 의미와 계산 로직 정의

3. **정규표현식 기반 광고 패턴 추출**
   - 대가성 문구 탐지 (예: "무료로 받았어요", "체험단으로...")
   - 홍보성 어미 탐지 (예: "~했답니다", "~해드립니다")
   - 기타 광고성 패턴 매칭

## 작업 대상 파일
- `core/validator.py` - 메인 검증 로직 모듈

## 활용 시점
- `core/validator.py` 작성 시
- 신뢰도 점수 공식 수정 및 검증 시
- 광고 판별 로직 개선 시
- 정규표현식 패턴 추가/수정 시

## 출력 요구사항
- 모든 코드에는 한국어 주석 포함
- 개발일지(`logs/dev_log.md`) 표준 양식 준수
- 테스트 케이스 작성 권장
- 예외 처리 로직 포함 (예: 리뷰 텍스트 10자 미만 제외)

## 참조 문서
- `SPEC.md` - 기획서 (13단계 체크리스트 참조)
- `logs/dev_log.md` - 개발일지 표준 양식
- `agents/interface.md` - 데이터 입출력 인터페이스 정의
